除錯(Debug)

既然len是用來詢問字串長度，那麼可不可以告訴我們數字有幾位數?
我們輸入len(189242)
應該會出現類似底下的訊息:

>>> len(189242)
Traceback (most recent call last):
File "< stdin>", line 1, in < module>
TypeError: object of type 'int' has no len()
欸!python到底在幹嘛啊... 不僅沒有告訴我們答案是6，甚至還跑出一堆奇怪的文字!

恭喜!恭喜!你看到了Python的第一則報錯，我們來一行行的分析Python到底在說些什麼吧!
TypeError是指類型錯誤，引號中的int指的是integer(整數)「int has no len()」指的是整數沒有長度， 導致python不知道怎麼告訴我們正確答案。

思考時間:189242很明顯是6位數對吧!那為什麼Python會認不出來呢?
這是因為Python不知道你寫的整數是10進位還是2進位，像是十進位的10會等於2進位的1010，而Python不能隨意假設你想用哪一種進位法， 所以就放棄了。

那怎麼辦?只要把數字轉換成字串就好了!
>>>len(str(189242))
6
我們在len函式當中，使用了str函式，str()可以把東西轉換成字串；這下len函式就知道該怎麼做了。
。str函式將東西轉換成字串
。int函式將東西轉換為整數
tips:我們可以把任何的數字轉換為字串，但不是所有的字串都能轉換為數字。如果輸入int('hi')會怎樣?

變數(Variable)

有時候，我們會想在很多地方使用同一個代號。但我們要怎麼告訴 Python，我們想要什麼東西?
為了讓彼此知道互相在講什麼東西，我們需要為這些東西取名字。這些名字就叫做變數，在Python中可以用等號=來創建變數:
>>>name 'E'
欸?Python怎麼沒有反應，難道是壞掉了嗎?
其實我們在Python建立變數時，Python只會默默地創建，不會回應。那我們要怎麼驗證呢?
>>>name
'E'
既然是變數，就表示可以改變內容。
>>>name = '1'
>>>name
'1'
而且變數是東西的名稱，所以可以把變數傳入函式內:
len(name)
1
當然也可以拿變數來計算，甚至可以把計算的結果設為變數:
>>>two = '2'
>>>three = '3'
>>>Ans = three * two
>>>Ans
6
我們來實驗看看輸入不存在的變數會怎樣:
>>>name = E
>>>nema
Traceback (most recent call last):
File "< stdin>", line 1, in < module>
NameError: name 'nema' defined
竟然又是一個錯誤!Python會根據你犯的錯誤，而反映出不同種類的報錯。我們稍早看過TypeError現在是NameError，指的名稱有誤。
